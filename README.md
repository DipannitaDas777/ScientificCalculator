# Scientific Calculator

##  Overview

Scientific Calculator is an Android application developed using Kotlin and Jetpack Compose. It provides both basic and scientific mathematical operations through a clean and user-friendly interface.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Percentage calculations
- Scientific functions (sin, cos, tan)
- Logarithmic functions (log, ln)
- Square root calculations
- Factorial support
- Exponent operator (^)
- Mathematical constants (π and e)
- Parentheses support for complex expressions
- Clear and Delete functionality
- Responsive and intuitive user interface

## Technologies Used

- Kotlin
- Jetpack Compose
- Android Studio
- Material Design 3

## How to Run

1. Clone this repository.
2. Open the project in Android Studio.
3. Allow Gradle to sync.
4. Run the application on an emulator or Android device.
